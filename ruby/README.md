# lolnews

A Ruby tool designed to download news headings about the game League of Legends.


# Usage

Run the application from command line:

    $ ruby lolnews.rb

Work interactively with the application via REPL:

    $ irb
    2.0.0p195 :001 > load './lolnews.rb'
